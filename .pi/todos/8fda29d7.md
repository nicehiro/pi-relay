## Decisions

- **Target channel**: sends to this machine's first configured channel, no arbitrary channel ID override
- **Trigger**: LLM calls `discord_send` during a conversation (not external/control socket)
- **Format**: plain text with simple markdown (Discord-native markdown passes through)
- **Independent of pendingChat**: works even when no Discord message triggered the session — sends to configured channel

## Implementation

- [x] `discord_send` tool registered with `{ message: string }` parameter
- [x] Sends to first configured channel, errors if not connected or no channel configured
- [x] Supports markdown, uses splitMessage for >2000 char messages
- [x] Works independently of pendingChat state
