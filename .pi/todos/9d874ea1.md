## Description

Add `promptSnippet` to the `discord_send` tool for a cleaner one-line entry in the system prompt's "Available tools" section.

## Current State

- `discord_send` has `description` but no `promptSnippet`
- The full `description` string is used in the system prompt's tool list

## Change

```typescript
pi.registerTool({
  name: "discord_send",
  label: "Discord Send",
  description: "Send a message to this machine's Discord channel",
  promptSnippet: "Send a message to this machine's Discord channel",
  // ...
});
```

Also for `spawn_session`:
```typescript
pi.registerTool({
  name: "spawn_session",
  label: "Spawn Session",
  description: "Spawn a new pi session in a tmux session with its own Discord thread",
  promptSnippet: "Spawn a new pi session in a Discord thread with its own working directory",
  // ...
});
```

Small change, can be done independently of other migration tasks.
